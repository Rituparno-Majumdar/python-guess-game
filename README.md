# Python Number Guessing Game

A simple interactive CLI game written in Python.

## How to play

```bash
python guess_game.py
```

1. Choose an upper limit (e.g. `100`)
2. The program picks a secret number in that range
3. Guess until you find it â€” you get hints after each attempt
4. Beat it within the allowed attempts to win!

## Features

- Smart attempt limit based on the chosen range
- Play again loop
- Clear feedback (too high / too low)

## Requirements

Python 3.6+, no external dependencies.
